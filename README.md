# Install
Clone repository:

```
$ git clone https://github.com/radosz99/pug-weather-service.git
```

Install `poetry` package:
```
$ pip install poetry
```
Install Python dependencies via poetry:
```
$ poetry install
```


# Run

By activating virtualenv:
```
$ poetry shell
$ uvicorn
```

Or via `poetry run`:
```
$ poetry run uvicorn
```
